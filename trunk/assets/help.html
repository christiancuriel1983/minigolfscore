Mini Golf Score 1.0.1, by Randall Spangler

Mini Golf Score is a free easy-to-use golf scorecard for Android. Source code is available at http://code.google.com/p/minigolfscore.

Quick Start

  • Set the number of holes and players in the Settings menu.
  • Tap on a hole number to set the current hole. The current hole is marked in orange.
  • Tap on a player to add a stroke for the current hole.
  • If you make a mistake, press Undo.
  • When you're done, you can E-mail your scores from the menu.

Configuring the Scorecard

  • Press the MENU button and select Settings. From here, you can set the number of players and holes for the scorecard. You can also set whether the scorecard is always displayed in landscape orientation, which lets you see more holes at once, and works best with 4 or fewer players.
  • Press and hold on a player's name to set that player's name.
  • Press and hold on a hole number to set par for that hole.
  • Tap on the Score in the upper right corner to toggle between showing scores as the total number of strokes, or the number of strokes above/below par.

Keeping Score

  • Tap on a player's name to add a stroke for that player on the current hole (which is highlighted in orange).
  • Tap Next hole to move to the next hole, or tap on a hole number to select that hole.
  • Tap on a player's score for a hole to edit that score.
  • Tap Undo to take back the last change (this also works to undo a change to par or a player's name).

Options Menu

Press the MENU button to display the options menu. From here, you can:
  • Clear the scores, so you can play again. (The players' names are not cleared, in case you're playing with the same people.)
  • E-mail the scores to yourself or your friends.
  • Change the Settings (see Configuring the Scorecard above).
  • Display this Help. (Press the back button to go back to the scorecard.)
